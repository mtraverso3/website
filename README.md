# Personal Website

My simple personal website. Hosted at [https://mtraverso.net](https://mtraverso.net).

## Quick start (local)

```bash
# install deps (npm/pnpm/yarn)
npm install

# run development server
npm run dev
```

Open http://localhost:3000
